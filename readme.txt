======================================
=======   Le Projet CinePose   =======
======================================

Il s'agit d'une application web employant le Java, basé sur le Framework Spring MVC.
Cette application web est destinée à classer les cinémas parisiens et rassembler les 
avis des utilisateurs. Elle fonctionne de manière collaborative, chaque utilisateurs
peut entrer des cinémas et laisser des avis. 

Ce projet à été développer dans le cadre d'un cours de Web Programming en 5ème année 
de l'ESIEAdispensé par Benoit Larroque.
